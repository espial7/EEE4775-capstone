EEL 4775 Real-Time Systems Capstone

Project Overview

This project demonstrates a real-time embedded system implemented using FreeRTOS on an ESP32. The system demonstrates real-time scheduling, inter-task communication, interrupt handling, synchronization, and real-time performance measurement.

System Architecture

The system consists of multiple FreeRTOS tasks running across the ESP32 cores. Tasks communicate using FreeRTOS synchronization and communication primitives.
